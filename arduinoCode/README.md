# Arduino Code of 2019-2020

This folder acts as the store for all the programs of used on the Arduino baords.

Hardware used:
 - Arduino boards
   - Arduino AVR Uno R3
   - Arduino AVR Mega 2560
 - Arduino Shields
   - Adafruit Motorshield v2
   - Adafruit 16-Channel Servo shield
 - Generic DC motors
 - SG90 Mini servos
